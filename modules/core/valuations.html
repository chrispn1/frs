In order to ensure correct pricing it is vital to ensure that this data is correct. However as funds increase in size and complexity this becomes harder and harder to manage manually. The exceptions system is designed to allow users specify rules that can be run before or after tasks to check that specified parameters are within specified bounds. Notifications are sent if an error is detected and errors can block subsequent processes until signed off. features: - Simple to use wizard for creating rules - adf benefits: - headline: Reduce risk detail: Through more regular and comprehensive checking of parameters.
